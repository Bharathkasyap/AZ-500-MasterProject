| Field | Details |
|---|---|
| **Exam Name** | Microsoft Azure Security Technologies |
| **Exam Code** | AZ-500 |

| **Exam Duration** | 150 minutes |

## 🎯 Key Azure Security Services — Quick Reference

| Category | Services |
|---|---|
| **Identity** | Microsoft Entra ID, Conditional Access, PIM, MFA, Azure AD B2C |
| **Networking** | NSG, Azure Firewall, DDoS Protection, VPN Gateway, Azure Bastion |
| **Compute** | Microsoft Defender for Cloud, Azure Security Center, Just-in-Time VM Access |
| **Storage** | Storage Account Encryption, SAS Tokens, Azure Key Vault |
| **Databases** | Transparent Data Encryption, Advanced Data Security, Azure SQL Auditing |
| **Operations** | Microsoft Sentinel, Log Analytics, Azure Monitor, Security Alerts |
