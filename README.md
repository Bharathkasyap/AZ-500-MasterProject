## ✅ Pre-Requisites

Before sitting the AZ-500, candidates should have working knowledge of:
- **AZ-900** (Azure Fundamentals) concepts
- **AZ-104** (Azure Administrator) skills: managing identities, compute, storage, networking
- Basic familiarity with PowerShell, Azure CLI, and Azure Resource Manager (ARM) templates

---

## 🔑 Key Skills Measured (Official Objectives Summary)

### Domain 1 — Manage Identity and Access
- Manage Azure Active Directory (Entra ID) identities
- Manage secure access by using Azure AD
- Manage application access in Azure AD
- Manage access control (RBAC, Azure Policy)

### Domain 2 — Secure Networking
- Plan and implement security for virtual networks
- Plan and implement security for private access to Azure resources
- Plan and implement security for public access to Azure resources

### Domain 3 — Secure Compute, Storage, and Databases
- Plan and implement advanced security for compute
- Plan and implement security for storage
- Plan and implement security for Azure SQL Database and Azure SQL Managed Instance

### Domain 4 — Manage Security Operations
- Plan, implement, and manage governance for security
- Manage security posture by using Microsoft Defender for Cloud
- Configure and manage threat protection by using Microsoft Defender for Cloud
- Configure and manage security monitoring and automation solutions
