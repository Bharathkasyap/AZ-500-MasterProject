## 📝 Exam Tips

- **Time management**: ~2.5 minutes per question. Flag difficult questions and return to them.
- **Performance-based labs**: Read all tasks before starting; changes persist across sub-tasks.
- **Case studies**: Read the requirements section first before reading the full scenario.
- **Elimination strategy**: Eliminate clearly wrong answers to improve odds on uncertain questions.
- **Know the defaults**: Many questions test default behaviors of Azure security services.
- **Hands-on practice**: A free Azure account with $200 credit covers all lab scenarios.
