## 🗓️ Study Schedule Template

### 4-Week Intensive Study Plan

**Week 1: Identity & Access (Domain 1 — 25–30%)**
- Day 1–2: Read Domain 1 study guide (this repo)
- Day 3–4: Complete Microsoft Learn — Manage identity and access path
- Day 5: Labs 1–4 (Conditional Access, PIM, Managed Identity, RBAC)
- Day 6: Domain 1 practice questions (Q1–Q18 from practice exam)
- Day 7: Review weak areas

**Week 2: Networking + Compute/Storage/Databases (Domains 2 & 3 — 40–50%)**
- Day 1–2: Read Domain 2 study guide + Microsoft Learn Networking modules
- Day 3: Labs 5–9 (NSG, Firewall, Bastion, Private Endpoint, WAF)
- Day 4: Read Domain 3 study guide + Microsoft Learn Secure Data path
- Day 5: Labs 10–13 (Key Vault, ADE, SQL, JIT)
- Day 6–7: Practice questions Domains 2 & 3 (Q19–Q44)

**Week 3: Security Operations (Domain 4 — 25–30%)**
- Day 1–2: Read Domain 4 study guide + Microsoft Learn Security Ops path
- Day 3–4: Labs 14–16 (Sentinel setup, analytics rules, Monitor alerts)
- Day 5: Explore Sentinel in Azure portal — create workbooks, hunting queries
- Day 6–7: Practice questions Domain 4 (Q45–Q60)

**Week 4: Review & Exam Ready**
- Day 1–2: Full practice exam (60 questions); identify weak areas
- Day 3: Deep review of weak domains using study guides
- Day 4: Re-read cheat sheet; review key CLI commands
- Day 5: Second full practice exam + focus on missed questions
- Day 6: Light review only; check in with cheat sheet
- **Day 7: EXAM DAY** 🎯

## 💡 Exam-Taking Tips

1. **Read questions carefully** — pay attention to qualifiers like "MOST cost-effective," "MINIMUM permissions," "WITHOUT storing credentials"
2. **Eliminate wrong answers first** — usually 1–2 answers are clearly wrong
3. **Watch for Entra ID P2 requirements** — many features require P2 specifically
4. **"Recommended" = Microsoft best practice** — usually means managed identity, RBAC, Private Endpoints
5. **Lab questions** — take your time; there's no penalty for reviewing steps
6. **Flag difficult questions** for review at the end — don't spend too long on one question
7. **The exam is in English** unless you specifically chose another language — bring photo ID
8. **Schedule early morning** if you're a morning person; the exam center environment matters
